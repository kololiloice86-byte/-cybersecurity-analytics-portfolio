# Cybersecurity & Data Analytics Portfolio – Loice Nasaka Kololi

Welcome to my professional portfolio. This repository demonstrates hands-on projects in **cybersecurity, data analytics, and reporting/visualization**. It highlights my ability to prepare statistical and operational reports, automate quality control checks, and design dashboards that “tell the story” behind the data.

## Objectives
- Deliver timely, accurate **standard and ad hoc reports**.
- Automate **quality control** checks and document training needs.
- Visualize **operations productivity** and trend insights for stakeholders.
- Maintain **compliance-ready** documentation aligned with ISO 27001, NIST, GDPR.

## Key Projects
- **SIEM Log Analysis (Splunk/Wazuh)** – Parse, clean, and classify events; generate incident summaries and KPIs.
- **Network Assessment (Nmap/Wireshark)** – Summaries of host discovery and traffic patterns.
- **SQL/SSRS Reporting** – Stored procedures, views, and functions powering standardized reports.
- **Dashboards (Tableau/Power BI/Excel)** – Resettlement ops productivity, QC outcomes, and training needs.

## Tech Stack
- Databases: **Microsoft SQL Server**, **Transact-SQL (T-SQL)**
- Reporting: **SSRS**, **Excel (advanced)**
- Visualization: **Tableau**, **Power BI**
- Scripting: **Python** (Pandas, NumPy)
- Tools: **Splunk**, **Wazuh**, **Wireshark**, **Nmap**
